# DesignPatternByCSharp

# Quick start

*Make sure that you have installed netcore sdk* [.NET Core](https://dotnet.github.io/)

>git clone https://github.com/Peefy/DotNetCore.DesignPatternByCSharp

>cd DotNetCore.DesignPatternByCSharp

>dotnet run
